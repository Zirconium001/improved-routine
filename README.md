# improved-routine

Improved Routine is a simple web project that displays the exact time and date of classes according to a routine.

The routine is stored in a JSON file, and JavaScript parses the JSON data to determine the correct class schedule based on the current day and time.

This project helped me successfully implement JSON parsing in JavaScript after several attempts.

## Features

- Shows class time according to the routine
- Uses JSON to store routine data
- Parses JSON using JavaScript
- Displays the correct class schedule based on date and time
- Simple and lightweight

## Built With

- HTML
- CSS
- JavaScript
- JSON

## Usage

1. The routine is stored in a JSON file.
2. JavaScript reads the JSON data.
3. The script parses the routine and determines the correct class based on the current day and time.
4. The webpage displays the current or upcoming class.


## Purpose

The goal of this project was to practice working with JSON data and learn how to parse JSON in JavaScript to build a dynamic schedule system.
